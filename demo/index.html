Hello AssetsDispatcher!

Assets Dispatcher makes it a little bit easier to work with versioned assets such as Javascript and stylesheets.

  • Deploy full size versions of your Javascript and stylesheet files.
  • Use AssetsDispatcher to minify and cache once on-the-fly for production.
  • Use versions in the asset's URL to easily deploy new versions. 
    <link href="./dispatcher.php/css/styles-v2.css" rel="stylesheet">
  • Use ?nocache to dispatch the original non-cached file for testing. 
    <link href="./dispatcher.php/css/styles-v2.css?nocache" rel="stylesheet">
